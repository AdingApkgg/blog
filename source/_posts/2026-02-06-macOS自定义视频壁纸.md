把自己的视频变成 macOS Tahoe 的动态壁纸，其实挺安静的一件事。

文件夹路径是  
`~/Library/Application Support/com.apple.wallpaper/aerials/videos/`

里面通常只有一两个 .mov 文件，比如  
`4C108785-A7BA-422E-9C79-B0129F1D5550.mov`（445MB 左右那种）。

步骤像这样：

1. 先去「系统设置 > 壁纸」，下载一个 Tahoe 的 Aerial 动态壁纸，让文件夹有内容。
2. 备份原 .mov 文件到别处（很重要）。
3. 如果你的视频是 webm，用 ffmpeg 转成 mov（视觉接近无损）：

```sh
ffmpeg -i input.webm -c:v libx264 -crf 18 -preset slow -pix_fmt yuv420p -c:a aac -b:a 192k output.mov
```

或者更“苹果味”的 ProRes：

```sh
ffmpeg -i input.webm -c:v prores_ks -profile:v 3 -c:a pcm_s16le output.mov
```

4. 把转好的文件重命名成和原文件一模一样的名字（比如 4C108785-... .mov）。
5. 替换进去。
6. 回到壁纸设置，选那个被替换的 Aerial，应该就能看到你的视频在锁屏和偶尔桌面动了。

桌面通常只播放一小段（省电），解锁或切换空间会再动一下。锁屏会更完整地循环。
如果黑屏，检查一下转出的 mov 是否能用 QuickTime 正常播放。H.264 + AAC 最稳。
慢慢试，不急……弄坏了还原备份就好。
深夜里调壁纸的感觉，其实挺温柔的，像给屏幕换了一段属于自己的呼吸。
